Available Chart Types

All AG Charts Community (free) chart types as reusable web components

Donut Chart

Perfect for showing proportional data with a modern donut style

Usage Example:

<ag-donut-chart id="chart" title="Sales by Category"></ag-donut-chart>

<script>
const chart = document.getElementById('chart');

// Option 1: Object array (standard)
chart.data = [
  { label: 'Electronics', value: 45 },
  { label: 'Clothing', value: 30 },
  { label: 'Books', value: 15 },
  { label: 'Home & Garden', value: 10 }
];

// Option 2: JSON string (auto-parsed)
chart.data = '[{"label":"Electronics","value":45}]';
</script>
